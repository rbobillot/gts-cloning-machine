# GTS-EVENT-MANAGER
---

Simple event manager for GTS
 - Websocket server
 - Http server

It must handle events (mostly async) linked to the GTS, between services:
 - [ ] NDS availability
 - [ ] gts-nds-flatpass availability
 - [ ] GTS to NDS Pokemon transfer
 - [ ] NDS to GTS Pokemon transfer
